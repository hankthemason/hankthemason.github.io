Hi! My name is Hank. I'm a software developer, recent Recurse Center alum, and New York City (Queens!) resident.

I spent my first five years in NYC working as a ballet accompanist, getting to play piano for some of the top ballet companies in the world. Gradually I began getting interested in programming because I wanted to build audio and MIDI-editing software.

Initially I was using Max/MSP, a visual programming language, but as my needs got more complicated, I shifted to using Javascript. Immediately I was hooked. I started learning how to build static web pages with Javascript and React, then started working on full-stack apps using Node.js/Express. Along the way I incorporated Typescript into my stack, along with libraries like D3.js and chart-js. I attended Recurse Center in early 2021, where I built several full-stack web apps, made some silly programs, and pair-programmed with other Recursers on complex algorithms.

More than anything else, I love programming because there's no end to the new things you can learn. I'm driven by a voracious curiosity and eagerness to try new things.
